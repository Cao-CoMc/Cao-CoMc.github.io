K2Cr2O7 Minecraft Server 网站文件（最终安全版本 - 方案3）
====================================================================

版本特性：
✅ 服务器 IP 完全隐藏，仅通过 QQ 群私发
✅ 模组列表直接嵌入 HTML（6分类，120+模组）
✅ 时间轴显示实时刷新时间（精确到秒）
✅ 数据保留策略：最近100条且不超过7天

时间显示说明：
- 图表X轴：显示时分秒（14:30:45）
- 时间轴列表：显示完整时间（2026-03-05 14:30:45）
- 悬停提示：显示完整日期时间
- 每次刷新网页都会新增一条记录

数据清理策略（方案3）：
1. 自动删除7天前的旧数据
2. 最多保留最近100条记录
3. 数据存储在浏览器 localStorage 中

文件结构：
- index.html    : 主页面（31KB，含完整模组列表）
- style.css     : 样式表（响应式布局）
- script.js     : JavaScript（实时时间记录）
- images/       : 可选文件夹，存放服务器截图

部署前必做：
1. 修改 script.js 第1行的 API 地址
   - 建议通过后端代理转发，不要直接暴露IP

2. 可选配置自动刷新：
   - 取消注释最后一行的 setInterval
   - 默认5分钟自动刷新一次

3. 创建 images/ 文件夹并放入截图（可选）

技术细节：
- 纯静态 HTML/CSS/JS，无需数据库
- 使用 Chart.js 绘制在线人数曲线
- 使用 localStorage 本地存储历史数据
- 所有外部链接使用 noopener noreferrer 防护
